Elementos de HTML e CSS

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.

CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web. CSS básico apresenta tudo que você precisa para começar. Responderemos a perguntas do tipo: Como mudo meu texto para preto ou vermelho? Como faço para que meu conteúdo apareça em determinados lugares na tela? Como decoro minha página com imagens e cores de fundo?

Principais Tags do HTML5

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

O elemento <main>

A área de conteúdo principal de um documento inclui conteúdo exclusivo desse documento e exclui conteúdo que é repetido em um conjunto de documentos, como links de navegação do site, informações de direitos autorais, logotipos e banners do site e formulários de pesquisa (a menos que a função principal do documento ou aplicativo seja a de um formulário de pesquisa).

O elemento <section>

O elemento section é usado para definir uma seção genérica de um documento ou aplicativo. Por exemplo, você pode optar por criar seções em torno de seu conteúdo; uma seção para informações de contato, outra seção para feeds de notícias, e assim por diante. É importante entender que esse elemento não é destinados a fins de estilização. Se você precisar envolver um elemento apenas para estilizá-lo, você deve continuar a usar um <div> como faria antes.

O elemento <nav>

O elemento <nav> é usado para encapsular os principais links de navegação para outras páginas ou partes dentro da mesma página. Como é para uso em grandes blocos de navegação, não é estritamente destinado ao uso em rodapés (embora possa ser) e afins, onde grupos de links para outras páginas são comuns. Se você costuma marcar seus elementos de navegação com uma lista não ordenada (<ul>) e um monte de tags de lista (<li>), você pode ser melhor servido com uma navegação e um número de tags <a> aninhadas.

O elemento <article>

O elemento de artigo é usado para agrupar um conteúdo independente. Ao estruturar uma página, pergunte se o conteúdo que você pretende usar em uma tag de artigo pode ser considerado como um todo e colado em um site diferente e ainda fazer todo o sentido. Exemplos óbvios de conteúdo que devem ser agrupados com um elemento de artigo são postagens de blog ou notícias. Esteja ciente de que, ao aninhar elementos de artigo, presume-se que os elementos de artigo aninhados estejam principalmente relacionados ao artigo externo.

O elemento <aside>

O elemento <aside> é usado para conteúdo relacionado tangencialmente ao conteúdo ao seu redor. Em termos práticos, costumo usá-lo para sidebars, ou conteúdo em uma pequena dica sobre um assunto relacionado em um post de blog. Também é considerado adequado para citações, publicidade e grupos de elementos de navegação. Basicamente, qualquer coisa não diretamente relacionada ao conteúdo principal funcionaria bem em um <aside>. Se fosse um site de comércio eletrônico, consideraria áreas como Clientes que compraram isso, também compraram... como principais candidatos a um <aside>.

O elemento <header>

Na prática, o <header> pode ser usado para a área de “manchete” do cabeçalho de um site, mas também como uma introdução a outro conteúdo, como uma seção de introdução dentro de um elemento de <article>. Você pode usá-lo quantas vezes forem necessárias na mesma página. Você pode ter um cabeçalho dentro de cada seção da sua página, por exemplo.

O elemento <footer>

Como o <header>, o elemento <footer> não faz parte do algoritmo de estrutura de tópicos, portanto, não seciona o conteúdo. Em vez disso, ele deve ser usado para conter informações sobre a seção em que se encontra. Pode conter links para outros documentos ou informações de direitos autorais, por exemplo. Assim como o <header>, ele pode ser usado várias vezes em uma página, se necessário. Por exemplo, pode ser usado para o rodapé de um blog, mas também como rodapé em um <article> de postagem de blog no mesmo blog. No entanto, a especificação observa que as informações de contato do autor de uma postagem de blog devem ser agrupadas por um elemento de endereço.

O elemento <div>

O elemento de agrupamento mais onipresente é o <div>. O <div> é tão amplamente usado porque não tem opinião. Não transmite nada. O único significado implícito de um <div> é que ele agrupa algo. Apesar disso, muitas vezes você verá um <div> envolvendo nada além de uma string de texto. Você só deve optar por um <div> como último recurso. É o elemento a ser usado quando você não consegue pensar em nada melhor.

O elemento <p>

O elemento <p> é usado para marcar um parágrafo. No entanto, não pense que isso significa que só pode ser usado em textos de 3 a 4 linhas. Pelo contrário, use-o para marcar qualquer texto que não possa ser melhor marcado com um dos outros elementos. Para texto não específico, o elemento <p> é definitivamente um melhor escolha do que um <div>.

CSS básico

Assim como o HTML, o CSS não é realmente uma linguagem de programação. Também não é uma linguagem de marcação — é uma linguagem de folhas de estilos. Isso significa que o CSS permite aplicar estilos seletivamente a elementos em documentos HTML.

Anatomia de um conjunto de regras CSS

p { 
     color: red;
}

Seletor (Selector): O nome do elemento HTML no começo do conjunto de regras. Ele seleciona o(s) elemento(s) a serem estilizados (nesse caso, elementos p). Para dar estilo a um outro elemento, é só mudar o seletor.

Declaração (Declaration): Uma regra simples como color: red; especificando quais das propriedades do elemento você quer estilizar.

Propriedades (Property): Forma pela qual você estiliza um elemento HTML. (Nesse caso, color é uma propriedade dos elementos <p>.) Em CSS, você escolhe quais propriedades você deseja afetar com sua regra.

Valor da propriedade (Property value): À direita da propriedade, depois dos dois pontos, nós temos o valor de propriedade, que escolhe uma dentre muitas aparências possíveis para uma determinada propriedade (há muitos valores color(cor) além do red(vermelho)).

Notas importantes sobre sintaxe:

  • Cada linha de comando deve ser envolvida em chaves ({}).
  • Dentro de cada declaração, você deve usar dois pontos (:) para separar a propriedade de seus valores.
  • Dentro de cada conjunto de regras, você deve usar um ponto e vírgula (;) para separar cada declaração da próxima.
p {
    color: red;
    width: 500px;
    border:1px solid black;
}

Diferentes tipos de seletores

Nome do seletor O que ele seleciona Exemplo
Seletor de elemento Todos os elementos HTML de determinado tipo. p Seleciona <p>
Seletor de ID O elemento na página com o ID especificado. Em uma determinada página HTML, é uma boa prática usar um elemento por ID (e claro, um ID por elemento) mesmo que seja permitido usar o mesmo ID para vários elementos. #my-id Seleciona <p id="my-id" > ou <a id="my-id">
Seletor de classe O(s) elemento(s) na página com a classe específicada (várias instâncias de classe podem aparecer em uma página). .my-class Seleciona <p class="my-class"> e <a class="my-class">
Seletor de atributo O(s) elemento(s) na página com o atributo especificado. img[src] Seleciona <img src="myimage.png"> mas não <img>
Seletor de pseudo-classe O(s) elemento(s) específicado(s), mas somente quando estiver no estado especificado. Ex.: com o mouse sobre ele. a:hover Seleciona <a>, mas somente quando o mouse está em cima do link.

Principais propriedades usadas no CSS

  • padding, o espaço ao redor do conteúdo (ex.: ao redor do texto de um parágrafo).
  • border, a linha sólida do lado de fora do padding.
  • margin, o espaço externo a um elemento.
  • width (largura de um elemento).
  • background-color, a cor atrás do conteúdo de um elemento e do padding.
  • color, a cor do conteúdo de um elemento (geralmente texto).
  • text-shadow: cria uma sombra no texto dentro de um elemento.
